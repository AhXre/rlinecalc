# Simple Calculator Application
This is a simple calculator aplication, the challenge is create a calculator that receives a string like `"2+1"` and successfully interprets it, executes the operation and prints the result

## Usage
The calculator usage is simple, for example, just run:
```bash
cargo run "123 + 2"
```
And the calculator will return the result.

The sintaxis for the calculator is:
```bash
cargo run <operation>
```
